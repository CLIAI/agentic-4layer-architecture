# L3 — Orchestration (Commands) — understanding

This file accumulates confirmed insights about the L3 layer **in this
project's context**, promoted from teach-back rounds or hand-written
during reflection.

L3 is the user-facing thin wrapper layer — `/slash-commands` that
delegate to an L2 agent. Commands should stay short (5–15 lines) and
carry no logic of their own. Entries here should state:

* what was learned / confirmed,
* the source of truth (code path or doc reference),
* the date of the round or note.
